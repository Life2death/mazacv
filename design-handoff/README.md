# Design Handoff

Drop the Claude-design output here:
- Exported HTML/CSS or React components (best), OR
- A style tile / spec doc, OR
- Screenshots (PNG) named per screen, e.g. `01-landing-mobile.png`, `02-scan-desktop.png`.

The implementing agent reads this folder for layout/visual detail and
`DESIGN_BRIEF.md` for copy + colour tokens.
